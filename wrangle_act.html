Summary of issues¶

Quality issues¶

archive_twitter dataframe¶
  1. Missing records (2356 instead of +5k : won't take into account here)
  2. Some retweets in the dataset
  3. Some tweets does not concern dogs(when text contains we only rate dogs, Please only send in dogs, etc)
  4. Useless columns for analysis : low data quality (in_reply_to_status_id, in_reply_to_user_id, source, retweeted_status_id , retweeted_status_user_id, retweeted_status_timestamp after deleting all retweet....)
  5. Erroneous datatypes(timestamp should be datetime)
  6. Invalid name for some rows( name like a, an,the, this, actually, getting, old, all,just,the, very)
  7. Invalid name for Eazy-E : Eazy-E instead of Eazy
  8. Incoherent values for rating denominator
  9. Missing values for dog_stage(pupper, dogger, floofer, puppo : some can be getted from the text column)
images_predicted dataframe¶
  1. Useless columns for analysis: low data quality (jpg_url,img_num)

Tidiness issues¶

archive_twitter dataframe¶
